# Nether Earth

Браузерный прототип реалтайм-стратегии в духе [ZX Spectrum-игры 1987 года](https://worldofspectrum.net/pub/sinclair/games/n/NetherEarth.tzx.zip) от David Stewart. На изометрической карте две фракции собирают роботов из шасси, оружия и электроники, захватывают нейтральные здания и сносят Central Computer друг друга.

Никакой сборки — три статических файла, открывается локальным http-сервером.

## Запуск

```bash
python3 -m http.server 8080
# или любой другой статик-сервер из этой же папки
```

Откройте `http://localhost:8080/` в современном браузере (Chrome/Firefox/Safari). Первая секунда уходит на декодирование SVG-спрайтов.

## Управление

- **WASD** / стрелки — двигать плавающую платформу игрока
- **ЛКМ по своей фабрике** — открыть панель постройки робота
- **ЛКМ по своему роботу** — выделить
- с выделенным роботом:
  - **ЛКМ по пустому тайлу** — приказ двигаться
  - **ЛКМ по вражескому юниту/зданию** — атаковать (в радиусе оружия)
  - **ЛКМ по нейтральному зданию** — пойти и захватить
- **ПКМ** — снять выделение
- **ЛКМ по экрану победы/поражения** — рестарт

## Механики

### Здания

| Здание             | HP  | Роль                                                       |
|--------------------|-----|------------------------------------------------------------|
| Central Computer   | 250 | Главный штаб. Его уничтожение — условие победы/поражения   |
| Factory            | 120 | Производит роботов                                         |
| Mine               | 70  | +1.0 материала/сек владельцу                               |
| Warehouse          | 80  | +0.4 материала/сек владельцу                               |
| Nuclear Reactor    | 90  | Декоративное (в оригинале давал энергию, здесь упрощено)   |

Захват: поставь своего робота на тайл с чужим/нейтральным зданием, подожди 2 сек — флаг меняется. Над зданием во время захвата показывается прогресс-бар.

### Робот = шасси + оружие + электроника

**Шасси** (HP / скорость в тайлах/сек / цена):

| Шасси     | HP | Speed | Cost | Особенности              |
|-----------|----|-------|------|--------------------------|
| Bipod     | 25 | 2.0   | 3    | Быстрый, хрупкий         |
| Tripod    | 45 | 1.5   | 5    | Баланс                   |
| Quad      | 70 | 0.9   | 8    | Танк                     |
| Anti-grav | 30 | 2.3   | 10   | Летает — проходит воду   |

**Оружие** (дальность в тайлах / урон / cooldown / цена):

| Weapon  | Range | Dmg | CD    | Cost | Примечание                  |
|---------|-------|-----|-------|------|-----------------------------|
| None    | —     | —   | —     | 0    | Только захват зданий        |
| Cannon  | 3     | 8   | 1.0 с | 2    | Баллистический снаряд       |
| Missile | 5     | 16  | 1.6 с | 4    | Самонаводящаяся ракета      |
| Phaser  | 4     | 22  | 1.3 с | 5    | Мгновенный луч              |

**Электроника** (множитель HP / цена):

| Brain   | HPx  | Cost | Поведение                                     |
|---------|------|------|-----------------------------------------------|
| Basic   | 1.0  | 0    | Слушается приказов, автоатакует в радиусе     |
| Nuclear | 1.5  | 3    | То же, но больше HP                           |
| AI      | 1.2  | 4    | Автономный: сам ищет ближайшую цель           |

Время постройки ≈ `сумма стоимостей × 0.45` секунд.

### Экономика

- Материалы копятся с захваченных шахт и складов
- Стартовые материалы: игрок — 30, противник — 35
- Энергия не моделируется (упрощение)

### ИИ противника

Каждые 5–8 секунд, если хватает материалов и есть свободная фабрика, враг строит AI-робота случайной конфигурации. Его роботы приоритизируют враждебные цели, а если никого нет в радиусе действия — идут захватывать ближайшее нейтральное здание.

## Стек

- Чистый JavaScript (ES2020), никаких фреймворков и бандлеров
- Canvas 2D для рендера
- Изометрическая проекция: `sx = (tx - ty) × 32`, `sy = (tx + ty) × 16`
- SVG-спрайты как строки → data URL → `<img>`. Тонировка по владельцу — подменой `__ACCENT__` на цвет фракции при загрузке (три варианта каждого тинтуемого спрайта)
- A\* на 4-связной решётке с манхэттенской эвристикой
- Painter's algorithm по глубине `tx + ty` + небольшие эпсилоны для слоёв (тайл → здание → юнит → платформа → снаряд)

## Структура

```
index.html    страница, HUD, панель-конструктор (HTML/CSS)
sprites.js    SVG-определения, SPRITES-реестр, тинтовый лоадер
game.js       константы, state, апдейт юнитов/зданий/снарядов, ИИ, ввод, рендер
```

## Отличия от оригинала

Прототип, поэтому срезано:

- Энергосеть / зависимость построек от реакторов
- Beeper-звук и музыка
- Точный баланс 1987 года — параметры тюнил на глаз
- Туман войны и радар
- Пиксель-арт — спрайты векторные, чистые, но без ретро-эстетики
- Уровни и сценарии — одна симметричная карта, один противник
